# ZK-Wordle

Wordle game implemented using the Zero-Knowledge Proofs.

## Compile the circuit, generate the reference zKey and verifier smart contract

```
npm install
npm run compile
```

## Run the backend

```bash
$> cd backend
$> npm install
$> npm start
```

## Run the frontend

```bash
$> cd frontend
$> npm install
$> npm run start
```
